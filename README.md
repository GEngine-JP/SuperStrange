#一. 本章阐述核心玩法且不动摇

核心设定如下

设定1，玩家会沿着公路一直往前跑

设定2，玩家每次跑酷都会被计时

设定3，分为多个关卡每个都有达成目标

设定4，在跑酷时间内分为昼夜交替

设定5，白天收集攻击道具/增益道具，晚上放出怪物(需要战斗)

设定6，一个攻击道具为一个“符号”

设定7，攻击方式为 玩家画出固定道具手势，如记忆成功，则释放攻击道具，反之可能面临攻击

设定8，角色具有 天赋技能 具有能量槽 具有血量

设定9，角色可以不适用白天积累的攻击道具，使用天赋技能，消耗能量
